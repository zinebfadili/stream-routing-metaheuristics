Running the program

To run the Simulated Annealing program, go to the simulated-annealing folder, open a command line and enter the following


java -jar simulated-annealing.jar <Path to input file> <Path to output file>


You need to specify the name of the file containing the network description in the first argument and the path you wish to put in the result file.


For Example:


java -jar simulated-annealing.jar TC0.xml TC0_solution.xml


Note :
In the inputs folder, there are the test cases provided.
In the solutions folder, there are the output that the Simulated Annealing gave for each test case.
